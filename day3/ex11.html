Responsive Navbar with Flexbox

This navbar uses d-flex, flex-column on mobile, and flex-md-row on tablets/desktops.

Navbar Features:
  • Collapses on mobile screens
  • Expands to horizontal layout on medium screens (md breakpoint)
  • Logo stays fixed while nav items align to the right (ms-auto)
  • Uses flexbox utilities for responsive alignment

How It Works:

The navbar uses Bootstrap's navbar component which internally uses flexbox utilities:

  • navbar-expand-md - Collapses on small screens, expands on medium and up
  • d-flex - Uses flexbox for the container
  • ms-auto - Pushes nav items to the right (margin-start: auto)
  • navbar-nav - Horizontal navigation list
