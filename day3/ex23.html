Bootstrap Spacing Utilities

1. Margin Classes

Margin adds space outside elements

m-2 (margin on all sides)
mt-4 (margin-top)
mb-3 (margin-bottom)
ms-5 (margin-start/left)
me-3 (margin-end/right)

2. Padding Classes

Padding adds space inside elements

p-2 (padding on all sides)
pt-3 pb-4 (padding top and bottom)
px-4 (padding left and right)
py-5 (padding top and bottom - large)

3. Margin Auto (Centering)

mx-auto (centered horizontally)

4. Responsive Spacing

Spacing changes based on screen size using breakpoint prefixes

m-1 (mobile) → m-sm-2 → m-md-3 → m-lg-4 → m-xl-5

5. All Spacing Sizes

Bootstrap supports 0-5 for most spacing utilities (with special values)

.m-0 (0 margin)
.m-1 (0.25rem margin)
.m-2 (0.5rem margin)
.m-3 (1rem margin)
.m-4 (1.5rem margin)
.m-5 (3rem margin)

6. Gap Utility (between flex/grid items)

g-0 (no gap)
g-0 (no gap)
g-2 (small gap)
g-2 (small gap)
g-4 (large gap)
g-4 (large gap)
Spacing Utility Reference
Class Value Description
m-* margin All sides
mt-*, mb-*, ms-*, me-* margin Individual sides
mx-*, my-* margin Horizontal/Vertical
p-* padding All sides
pt-*, pb-*, ps-*, pe-* padding Individual sides
g-* gap Between flex/grid items
